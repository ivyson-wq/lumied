Politica de Privacidade

Lumied CRM - WhatsApp Templates (Extensao para Google Chrome)

1. Informacoes gerais

Esta Politica de Privacidade descreve como a extensao Lumied CRM - WhatsApp Templates (doravante "Extensao") coleta, utiliza e protege os dados dos usuarios. A Extensao e desenvolvida e mantida pela Lumied (lumied.com.br).

2. Dados coletados

A Extensao NAO coleta, armazena ou transmite dados pessoais dos usuarios para servidores externos de terceiros.

Os unicos dados armazenados localmente no navegador do usuario (via chrome.storage.local) sao:

  • URL do sistema — endereco do servidor Supabase da escola
  • API Key — chave publica (anon key) do projeto Supabase
  • Token de autenticacao — token de sessao do gerente escolar

Esses dados sao inseridos manualmente pelo proprio usuario e permanecem apenas no dispositivo local, nunca sendo compartilhados com terceiros.

3. Dados acessados

A Extensao acessa:

  • Templates de mensagens CRM — buscados do servidor Supabase da propria escola do usuario, mediante autenticacao
  • Campo de texto do WhatsApp Web — para inserir o template selecionado na conversa ativa (somente quando o usuario clica em "Enviar no chat")

A Extensao NAO le, armazena ou transmite:

  • Mensagens de conversas do WhatsApp
  • Contatos ou listas de contatos
  • Midias, fotos ou documentos
  • Historico de navegacao
  • Dados pessoais de terceiros

4. Permissoes utilizadas

  • storage — para salvar as credenciais de configuracao localmente
  • activeTab — para interagir com a aba ativa do WhatsApp Web quando o usuario clica em enviar template
  • host_permissions (web.whatsapp.com) — para injetar o painel de templates na interface do WhatsApp Web

5. Compartilhamento de dados

A Extensao NAO compartilha nenhum dado com terceiros. As comunicacoes ocorrem exclusivamente entre o navegador do usuario e o servidor Supabase da escola onde o usuario esta autenticado.

6. Seguranca

Todas as comunicacoes entre a Extensao e o servidor da escola sao realizadas via HTTPS com criptografia TLS. As credenciais sao armazenadas na area segura do Chrome (chrome.storage.local), acessivel apenas pela propria extensao.

7. Retencao de dados

Os dados de configuracao permanecem armazenados localmente ate que o usuario:

  • Desinstale a Extensao
  • Limpe os dados da extensao nas configuracoes do Chrome
  • Altere manualmente as configuracoes no popup da extensao

8. Direitos do usuario (LGPD)

Em conformidade com a Lei Geral de Protecao de Dados (Lei 13.709/2018), o usuario pode:

  • Acessar seus dados — todos estao visiveis no popup de configuracao
  • Corrigir seus dados — editando os campos no popup
  • Excluir seus dados — desinstalando a extensao ou limpando os dados
  • Revogar consentimento — desinstalando a extensao a qualquer momento

9. Uso por menores

A Extensao e destinada exclusivamente a profissionais de gestao escolar (gerentes, coordenadores, secretarias). Nao e direcionada a menores de 18 anos.

10. Alteracoes nesta politica

Esta politica pode ser atualizada periodicamente. A versao mais recente estara sempre disponivel nesta pagina e na listagem da Chrome Web Store.
