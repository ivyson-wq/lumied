📧 E-mail (Resend)

Envio de e-mails de ausências, turnos, atividades e acesso.

Domínio verificado no Resend. Remetente: noreply@domínio
Recebe avisos de ausência, turnos, inscrições
Configurar Resend:
1. Crie conta em resend.com
2. Verifique o domínio acima (DNS)
3. Gere uma API Key e configure no Supabase:
Copie para: Supabase > Edge Functions > Secrets > RESEND_API_KEY

🛒 Mercado Livre (Almoxarifado)

OAuth para busca de preços. Opcional — almoxarifado funciona sem ML.

developers.mercadolivre.com.br > Criar app
Cadastre esta URL como redirect no app do ML
Secrets no Supabase: ML_CLIENT_ID, ML_CLIENT_SECRET

🏦 Banco Inter (Boletos)

mTLS para boletos e consulta de pagamentos. Requer relay no Render.

URL do relay mTLS no Render
Supabase secrets: INTER_CLIENT_ID, INTER_CLIENT_SECRET, INTER_CONTA, INTER_RELAY_URL, RELAY_SECRET
Render env vars: RELAY_SECRET, INTER_CERT (PEM), INTER_KEY (PEM)

🔑 Google (Maps + Calendar)

Google Cloud Console > APIs & Services
Para Google Calendar
Secrets: GOOGLE_MAPS_KEY, GOOGLE_SERVICE_ACCOUNT
