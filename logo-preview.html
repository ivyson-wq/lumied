Lumied — 3 Conceitos Diferentes

Conceito A: Nodes / Rede  •  Conceito B: Nascer do Sol  •  Conceito C: Badge / App Icon

Conceito A — Nodes / Rede Conectada

L formado por pontos conectados. Transmite: rede, conexão entre portais, tecnologia. Paleta: indigo + ciano.

Conceito B — Nascer do Sol / Horizonte

Sol nascendo sobre horizonte com livro sutil. Transmite: novo dia, iluminação, crescimento. Paleta: âmbar + coral + violeta.

Conceito C — Badge / App Icon

Quadrado arredondado com "L" recortado. Funciona como app icon. Transmite: solidez, confiança, profissional. Paleta: verde + azul.
